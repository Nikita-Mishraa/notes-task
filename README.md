# Notes Task Manager

A full-stack Notes & Task Management application built with the MERN stack. The application allows users to create, organize, update, and delete notes and tasks through a clean and responsive interface.

## Features

- User Authentication
  - Secure login and registration
  - JWT-based authentication
  - Protected routes

- Notes Management
  - Create notes
  - Edit existing notes
  - Delete notes
  - View all notes

- Task Management
  - Create tasks
  - Update task status
  - Delete tasks
  - Organize daily activities

- Responsive Design
  - Mobile-friendly interface
  - Clean and modern UI

## Tech Stack

### Frontend
- React.js
- Axios
- React Router
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Tokens)
- bcrypt.js
